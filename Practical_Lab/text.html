Types of Text

  1. Formatted Text-Formatted text is text that is displayed in a special, specified style. In computer applications, formatting data may be associated with text data to create


  2. Unformatted Text-Unformatted text is any text that is not associated with any formatting information. It is plain text, containing only printable characters, white space, and line breaks


  3. HyperText-Hypertext is text which contains links to other texts
